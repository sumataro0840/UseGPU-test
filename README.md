# GPUを使っていろいろしたい旨

## 背景

WSL2 + Docker + CUDA + PyTorch の環境構築が完了し、GPU が正常に認識されることを確認した。  
実際に PyTorch から `cuda:0` 上でテンソル演算が実行され、行列サイズを大きくした場合には CPU より GPU の方が大幅に高速であることも確認できた。

そのため、今後は単なる環境確認に留まらず、GPU を使ってさまざまな処理や実験を行える状態を整えたい。

---

## 現時点の方針

現状では、特定の用途を一つに決め打ちするというよりも、まずは GPU を使って何ができるのかを広く試したいと考えている。  
そのうえで、自分にとって面白いテーマや、継続して触りたい分野を見つけたい。

特に、以下のような方向には関心がある。

- PyTorch を用いた数値計算や機械学習
- 大規模な行列演算や並列計算の体験
- ノイズ除去や信号処理のような、工学寄りのテーマ
- CUDA を用いた GPU 計算の基礎理解
- 将来的なローカル LLM 推論や推論高速化の検証

---

## 今やりたいこと

今すぐ明確なプロダクトや研究テーマがあるわけではないが、せっかく GPU 環境が動くようになったので、まずは「GPU を使うと何が嬉しいのか」を自分の手で確かめたい。

そのため、当面は以下を目標にする。

1. GPU で実行すると効果の大きい処理をいくつか試す  
2. CPU と GPU の違いを体感しながら理解する  
3. 機械学習に限らず、信号処理・数値計算・推論なども候補に入れる  
4. 面白そうな用途が見つかれば、その方向でフォルダ構成やコードを整理していく  

---

## 補足

Kaggle のように既存データセットを与えられてスコアを競う形式には、現時点ではあまり強い興味はない。  
それよりも、

- 自分でデータを作る
- 自分で問題設定を決める
- GPU を使う意味がある処理を試す

といった進め方の方が、自分には合っていそうだと感じている。

---

## まとめ

GPU 環境はすでに使える状態になっている。  
今後は、用途を最初から狭く決めすぎず、GPU を使ったさまざまな実験や処理を試しながら、自分にとって面白いテーマを探していきたい。
